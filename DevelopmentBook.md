| **Day 2** | **Core Backend (Auth)** | Create `User.js` Mongoose Model, implement `auth.js` routes (`/register`, `/login`), write `authController.js` logic (bcrypt, JWT), implement JWT Middleware. | 8 hrs |
| **Day 3** | **Core Frontend (Auth)** | Set up React project (`main.jsx`, `App.jsx`), create `AuthContext.js`, develop responsive `LoginForm.jsx` and `RegisterForm.jsx` using Tailwind and the provided color palette. | 8 hrs |
| **Day 4** | **Personalization & Onboarding** | Create `PreferencesForm.jsx` to capture user triggers/methods. Update `User.js`model to store these preferences. Finalize `UserProfile.jsx` to display data. | 8 hrs |
| **Day 5** | **Meditation Backend API** | Create `Meditation.js` Model and seed initial data. Implement `meditations.js`routes (`/get-all`, `/get-recommended`). Implement basic recommendation logic. | 8 hrs |
| **Day 6** | **Meditation Frontend (Browse)** | Develop `MeditationCard.jsx` and `MeditationLibrary.jsx`. Display all available and recommended meditations on the main "Browse" page. Implement search functionality. | 8 hrs |
| **Day 7** | **Meditation Player (Core)** | Develop `MeditationPlayer.jsx`. Implement play/pause, duration options (5/15/30 min), and transcript display. Implement a simple "session started" API call. | 8 hrs |
| **Day 8** | **Progress Tracking (Logs)** | Create `ProgressLog.js` and `MoodLog.js` models. Implement API endpoints for session completion and daily mood check-in. | 8 hrs |
| **Day 9** | **Progress Dashboard (MVP)** | Develop `StatisticsCard.jsx` and `ProgressDashboard.jsx`. Display basic stats: Total Sessions, Total Time, and a simple Streak Counter (data read from `ProgressLog`). | 8 hrs |
| **Day 10** | **Final Touches & Testing** | Review all components for Calming UX/UI (Color, Typography, Animations). Implement routing (`ProtectedRoute.jsx`, `NotFound.jsx`). Write basic unit tests for critical API routes. | 8 hrs |
